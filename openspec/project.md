# Project context - atender-application-engine

The application IS the demonstration. This repo is an agent-orchestrated engine that produces the
artifacts of a job application (a video, a portfolio patch, a voiceover) the same way its author
ships production software: spec-driven, fleet-built, gated by a tested contract in CI.

Voice: plain English, hard numbers, no emoji. Design tokens mirror the portfolio DESIGN.md
(cream #F1EAD9, ink #1A1612, terracotta #C25B2D, Fraunces + JetBrains Mono).
